## Vue d'ensemble

**FAKT** est une application desktop (Windows, macOS, Linux) conçue pour les freelances
et petites agences francophones qui veulent gérer leur cycle de facturation en local,
sans envoyer leurs données à un service cloud tiers.

**En une ligne :** Yousign + Indy + Google Drive fusionnés en un binaire de ~8 Mo,
open-source, hors-ligne par défaut.

## Trois modes de déploiement

FAKT suit le pattern « outil interne AlphaLuppi » :

| Mode | Version | Description |
|------|---------|-------------|
| **Solo local** | v0.1+ | Binaire desktop, SQLite, tout reste sur votre machine |
| **Self-host entreprise** | v0.2 | Docker Compose, backend partagé, multi-utilisateurs |
| **SaaS hébergé** | v0.3+ | On vous gère tout, ~12 €/user/mois |

## Conformité légale française

FAKT intègre les contraintes légales françaises non-négociables :

- Numérotation séquentielle sans trous (CGI art. 289).
- Mentions légales obligatoires sur facture (SIRET, pénalités retard, indemnité 40 €).
- Mention TVA non applicable art. 293 B CGI (micro-entreprise).
- Archivage 10 ans — aucune suppression physique des factures émises possible.
